Other tools

During the e-cidadania development we have created some scripts and tools that you may find useful not just during the ec-idadania development but in other django projects as well.

The majority of this tools are little scripts that make easier the maintenance of a big platform like, for example, a script that takes the documentation from the repository (previously multilanguage) and after that compiles it and puts it on the server in three formats each language. In the same way another script takes all the language catalogs from your applications and makes/compiles them, that way you won't need a central language catalog.
