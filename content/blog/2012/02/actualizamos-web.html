--- title: Actualizamos la web created: !!timestamp '2012-02-09 10:00:00' --- {% mark excerpt %} Hemos actualizado la web de e-cidadania para que sea más moderna y acorde al estilo de la propia plataforma. Creemos que este cambio ha sido para bien, aunque la anterior web cumplía muy bien su cometido, se nos había hecho un poco pequeña al no poder poner noticias en uin blog, o extender mejor las secciones de la misma. Como todo lo relacionado con e-cidadania, esta web también tiene su código disponible en el portal de GitHub, donde podréis dejarnos también las incidencias que veais. {%- endmark %} Como todo lo relacionado con e-cidadania, esta web también tiene su código disponible en el portal de GitHub, donde podréis dejarnos también las incidencias que veais.
