Plataforma completa

e-cidadania é o primeiro esforzo por intentar levar un proceso participativo completo combinando participación en liña con presencial. Dende os primeiros debates, pasando polas propostas e votacións, todo se fai dentro da propia plataforma, sen ter que andar de un sitio a outro.

Sistema de debates ordenado

e-cidadania inclé un novo sistema de debates para deixar atrás os sistemas lineais de discusión e hora de chegar a conclusións!.

Software libre

Con todas as súas letras. Toda a plataforma, dende o código fonte, pasando polo deseño, documentación, ferramentas auxiliares, etc. está licenciado baixo GPLv3 para garantir a sua liberdade perpetua.
